#Impolex G on the Node
Impolex G Prototyping is a simple prototyping tool built on a Javascript stack.

##Installation

```
git clone git@github.com:plechazunga/imipolex-g.git
npm install
gulp
```

##Features
- BrowserSync
- Basic Gulp Tasks
- Dead simple Hapi.js Server
- React as Partials
- Layouts in Jade
- Browserify
